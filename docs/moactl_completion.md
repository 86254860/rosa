## moactl completion

Generates bash completion scripts

### Synopsis

To load completion run

. <(moactl completion)

To configure your bash shell to load completions for each session add to your bashrc

# ~/.bashrc or ~/.profile
. <(moactl completion)


```
moactl completion [flags]
```
